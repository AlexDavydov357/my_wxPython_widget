# my_wxPython_widget

Набор кастомных виджетов для **wxPython**.  
Включает спиннеры, слайдеры, крутилки, всплывающие окна и прочие GUI-элементы.

## Установка

```bash
# Установить как обычный пакет
pip install my-wxpython-widget

# Или в режиме editable (для разработки)
git clone https://github.com/AlexDavydov357/my_wxPython_widget.git
cd my_wxPython_widget
pip install -e .
